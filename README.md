## Material Ocean Nomad

Material oceanized theme with nomad icons for KDE plasma

## included

- desktop theme
- color scheme

## Installation:

#### From source:

- `git clone https://github.com/material-ocean/Plasma.git`
- `cd Plasma`
- `sh install.sh`
